## Установка

```bash
npm install
```

## Запуск

### 1. Запуск сервера
```bash
node game-server.js
```

Сервер запустится на порту 5555 и будет ожидать подключений от клиентов.

### 2. Запуск клиента
```bash
node game-client.js <min> <max>
```

Где:
- `<min>` - минимальное значение диапазона
- `<max>` - максимальное значение диапазона
